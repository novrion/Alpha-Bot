# Beta Bot

**Tutorial**

1: input w/n for player side

2: input max time for engine to calculate next move (seconds)

3: input fen code or 'x' to start a game

4: write moves to play them. Ex: "e2e4" or "b1c3"




**Specifications**

Bitboard representation

Negamax

Alpha beta pruning

Null move heuristic

LVVMVA move order
