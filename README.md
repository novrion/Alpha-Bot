# Beta Bot

**Tutorial**

1: input w/n for player side\n
2: input max time for engine to calculate next move (seconds)\n
3: input fen code or 'x' to start a game\n
4: write moves to play them. Ex: "e2e4" or "b1c3"\n

**Specifications**
Bitboard representation
Negamax
Alpha beta pruning
Null move heuristic
LVVMVA move order
